## Description

System management has evolved from packages, to configuration management, and now CRDs. We keep trying to automate Linux with new tools and interfaces, but the underlying interface doesn't change. Shell scripts still rule the world and dominate workflows. When things go wrong, SSH is still the de facto standard. But what would Linux look like if it were built in today's world without legacy baggage. This talk will explain the evolution and lessons learned from building an API for Linux used in Talos Linux. A completely API managed distribution without any shells or SSH.
