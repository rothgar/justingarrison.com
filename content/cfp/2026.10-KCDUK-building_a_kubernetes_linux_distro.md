## Description

What if you built a Linux distro that only worked with Kubernetes?
It could have an API instead of a shell, it could be configured the same way as Kubernetes, and it could handle common administration tasks with declarative config.
Talos Linux was created to eliminate duplicate work, secure underlying infrastructure, and safely maintain Kubernetes and Linux together.
This talk will show how Talos works and why API driven distributions are the future of Linux.
